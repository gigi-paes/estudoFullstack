Descrição
Um site simples que lista receitas de padaria a partir de um arquivo JSON.
O usuário pode visualizar detalhes das receitas, como ingredientes e modo de preparo.
Desenvolvimento focado em manipulação de JSON, DOM e layout responsivo.

Funcionalidades
Exibe lista de receitas
Busca/filtragem
Detalhes de cada receita
Layout responsivo
Dados carregados a partir de JSON externo

Tecnologias usadas
HTML
CSS
JavaScript
JSON
Google AI Studio
Node.JS
